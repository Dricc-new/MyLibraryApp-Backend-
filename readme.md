# My Library (Backend)
This project is the API of a web of a library.

## Routes
| Route      | Method | Second Header 
| -----------|--------| ------------- 
| /tasks     | GET    | Returning all books on database  
|            | POST   | create a book on database 
| /tasks/:id | GET    | return a book by its id 
|            | PUT    | update a book  
|            | DELETE | remove a book
